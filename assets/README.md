## ğŸ¨ Asset Guidelines

### Screenshots
- **Resolution**: Minimum 1920x1080 for dashboard screenshots
- **Format**: PNG for clarity, JPG for photos
- **File Size**: Optimize to < 500KB per image

### Branding
- **Color Scheme**: Professional blues and greens
- **Consistency**: Match Power BI theme across all dashboards
- **Quality**: High-resolution, crisp images

### Banner Images
- Dimensions: 1200x300 pixels recommended
- Include portfolio title and key skills
- Professional and clean design

## ğŸ“¸ Screenshot Best Practices

When capturing dashboard screenshots:

1. **Clean the workspace**
   - Remove any test data or personal information
   - Ensure all visuals are fully rendered
   - Hide unnecessary panes or ribbons

2. **Highlight key features**
   - Show interactive elements
   - Display meaningful data
   - Include filters and slicers

3. **Quality checks**
   - No pixelation or blur
   - Proper aspect ratio
   - Readable text at preview size

4. **Naming convention**
   - Use descriptive names: `sales-dashboard-overview.png`
   - Include project name in filename
   - Keep filenames lowercase with hyphens
